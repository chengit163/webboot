# webboot
Admin管理后台（基于SpringBoot的学习及二次开发框架）

### 基本功能
1. 菜单管理  
2. 角色管理  
3. 用户管理  
4. 代码生成器  
5. Swagger  
6. 任务计划  
7. 日志管理  
8. Druid Monitor  
9. 在线管理  

![webboot](screenshots/webboot.png)
